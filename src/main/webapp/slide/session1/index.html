Heroku

Platform as a service (PaaS) that enables developers to build, run, and operate applications entirely in the cloud.

heroku.com

Crea nuova applicazione

  • git clone https://github.com/bayois/mineSQL new-name
  • cd new-name
  • heroku create new-name
  • git push heroku master
  • heroku open
Nuovo repository GIT creato automaticamente
  • heroku https://git.heroku.com/boma-0.git (fetch)
  • heroku https://git.heroku.com/boma-0.git (push)

Run Local

  • mvn package
  • heroku local
