Creazione nuova applicazione

come nasce BOMA

  • git clone https://github.com/bayois/mineSQL boma
  • cd boma
  • heroku create boma
  • git push heroku master
    avvia deploy
  • heroku open
    istanza cloud
Repository GIT Heroku

viene creato automaticamente

  • heroku https://git.heroku.com/boma-0.git (fetch)
  • heroku https://git.heroku.com/boma-0.git (push)
Repository GIT Origin
  • origin https://github.com/bayois/mineSQL (fetch)
  • origin https://github.com/bayois/mineSQL (push)

git push heroku origin
avvia continuos integration e deploy

Start Locacal Server

sviluppo e test

  • mvn package
  • heroku local
  • firefox http://localhost:5000
    la porta 5000 é un requisito di heroku
